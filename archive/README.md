# Archive

This folder contains older implementations, design demos, and planning artifacts that are useful for reference but are not active site entrypoints.

## Contents

- `legacy-backend-static-stock/` - old backend-served stock UI that previously caused confusion at the backend root.
- `legacy-react-stock-frontend/` - separate minimal React/Vite stock app.
- `design-and-planning/` - demos, mockups, and planning docs from earlier iterations.

Active project code remains in the repo root project folders.
