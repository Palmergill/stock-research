# Frontend

React + TypeScript + Vite + Recharts

## Setup

```bash
npm install
```

## Run (dev)

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Dev workflow

1. Start backend: `cd ../backend && uvicorn app.main:app --reload`
2. Start frontend: `npm run dev`
3. Open http://localhost:5173
