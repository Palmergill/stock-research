Projects

Stock Research

Stock Research is a tool for digging into real-time market data, company fundamentals, and financial trends in one place. I built it to make comparing companies faster and more visual, with interactive charts and useful metrics close at hand. It uses FastAPI on the backend and a lightweight vanilla JavaScript interface on the frontend.

FastAPI PostgreSQL Canvas Charts

Texas Hold'em Poker

Texas Hold'em Poker is a mobile-friendly No-Limit Hold'em game where you can play against AI opponents. The project focuses on realistic table flow, hand evaluation, and bot decision-making that makes each hand feel a little different. It has been a fun way to combine game design, backend logic, and AI behavior.

FastAPI AI Opponents Mobile First

Craps

Craps is a single-player Vegas-style dice game with authentic table rules and betting options. You can place Pass Line, Come, and proposition bets while the game handles the point, payouts, and dice results. I built it as a polished browser game that captures the rhythm of a casino table without needing a download.

Vanilla JS Vegas Rules Dice Game

Bitcoin Chat

Bitcoin Chat lets you ask plain-English questions about blocks, transactions, mempool fees, and node status. It connects chat-style interaction to read-only blockchain data so technical Bitcoin details are easier to explore. The goal is to make node information feel approachable without hiding the underlying data.

FastAPI Bitcoin Core Chat UI
